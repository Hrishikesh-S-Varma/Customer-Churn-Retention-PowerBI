## Project Overview
This project focuses on understanding customer churn behavior and identifying key factors that influence customer retention.

## Business Problem
Customer churn directly impacts revenue. Identifying churn drivers helps businesses improve retention strategies and reduce revenue loss.

## Key KPIs
- Churn Rate
- Retention Rate
- Customer Lifetime Value (CLV)
- Monthly Churn Trend
- Segment-wise Churn %

## Dashboard Features
- Customer segmentation analysis
- Churn trend visualization
- Behavioral and demographic breakdowns
- Interactive filters for deep-dive analysis

## Tools Used
- Power BI
- Power Query
- DAX
- Excel
- GitHub

## Outcome
The dashboard provides actionable insights to improve customer retention and reduce churn risk.
